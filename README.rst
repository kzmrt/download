
=======================
Download
=======================

This is CSV data download sample project.


Requirement
===========

:Python: 3.7.0
:cycler: 0.10.0
:Django: 2.2.4
:kiwisolver: 1.1.0
:matplotlib: 3.1.1
:numpy: 1.17.0
:pyparsing: 2.4.2
:python-dateutil: 2.8.0
:pytz: 2019.2
:six: 1.12.0
:sqlparse: 0.3.0


Quick start
===========
1. clone project::

    git clone https://github.com/kzmrt/download

2. migrate model::

    python manage.py migrate

3. create superuser::

    python manage.py createsuperuser

4. start server::

    python manage.py runserver
